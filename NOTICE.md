# Provenance Notice

This runtime-backed Grok Build plugin was adapted from the owner's Codex implementation and the earlier Claude/Grok Relentless Inception design lineage. Workflow ideas were also informed by the owner's Batch Create Eval, Gigaprompt, Exaflop, adversarial-review, and TrustedRouter/OpenRouter fusion research. Those ideas were re-expressed for Grok Build; source text was not copied wholesale.

Release-campaign jigs and curated receipts are published separately in `ahuserious/grok-fusion-artifact` at tag `limited-cost-2026-07-20`. That artifact intentionally excludes credentials, auth state, private Grok sessions, encrypted update material, raw prompt history, local configuration, and absolute private paths.

The source lineage did not include a selected distribution license. This repository therefore does not infer or silently assign one. The owner should choose an explicit license before granting public reuse rights.
